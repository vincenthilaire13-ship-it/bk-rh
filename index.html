📲 QR code de candidature

Un seul QR code pour les trois restaurants. Le candidat le scanne, arrive directement sur la fiche de recrutement et choisit lui-même le restaurant souhaité — sans code ni accès à l'application.

L'affiche PDF contient le QR code et votre phrase d'accroche, prête à imprimer ou envoyer (mail/WhatsApp).
